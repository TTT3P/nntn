13 ถุง ไม่ต้องพิมพ์ 13 ครั้ง

  • "fill ☰" uniform-weight (commit a471f97 · 21/04) — ชั่ง 1 ถุงได้ค่า 520 g → กด fill ลงอีก 12 ถุง · reduce error แล้วน้องยังแก้ได้ทีละถุงถ้าต้องการ
  • คลัง default = C — ถุงออกจากครัวไปคลัง C (317 bags ready-to-deliver) · tap เปลี่ยนได้ถ้าจะเก็บคลัง B freezer ก่อน
  • Multi-SKU output group — MT-031 จะมี 2+ groups (commit ffeebb4) · ทุก group ส่งเป็น element ใน p_produced: [{item_id, bags: [...]}]
  • Yield recon ใน UI — จาก v_production_reconciliation เทียบ recipe standard (82%) กับ actual (80.96%) · ±3% ปกติ · >5% แสดง warning "ลืมชั่งถุงไหนหรือเปล่า?"
  • Next ปุ่มเขียน "ดูสรุป" — ไม่ใช่ "บันทึก" ตรง · ขั้นยืนยันแยก เพื่อ double-check ก่อน 13 ถุงเข้า DB ผิด (ดัก incident 22/04)
